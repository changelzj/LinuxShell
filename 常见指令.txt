

-- 关机/重启
shutdown -h now 立即关机

shutdown -h 1 1分钟后关机

shutdown -r now 立即重启

halt 停机，就是关机

reboot 重启


sync 将内存数据同步到磁盘上（关机重启前最好执行一次,防止数据丢失）

-- 用户登录注销

logout 退出，图形界面无效只能退出终端




-- 实用指令

- 7个指定运行级别：

0：关机
1：单用户（找回丢失密码）
2：多用户无网络服务
3：多用户有网络
4：保留级别，暂未启用
5：图形化界面
6：重启

运行级别配置文件是 /etc/inittab
命令：init 级别数字



-帮助

man ls :查看ls命令有哪些用法

help cd：查看cd命令的使用方法

-- 文件目录指令

pwd 当前目录的绝对路径

ls [选项] 目录或文件
ls -a 显示所有文件和目录，包括隐藏的
ls -l 以列表形式显示

cd ~ 和 cd ：回到家目录

cd .. 回到当前目录的上一级目录

mkdir 创建目录
mkdir -p 创建多级目录

rmdir 删除目录下没有内容的空目录

rm  删除
rm -f 强制删除
rm -r 递归删除文件夹 
rm -rf 删除非空目录


创建文件，支持多个：touch 文件名 文件名 ......  ： 创建空文件

拷贝文件： cp 源文件 目标文件
cp -r 递归复制整个文件夹
\cp 复制过程中强制覆盖


移动或重命名：mv
重命名： mv 文件名  文件名
移动： mv 文件名 路径

cat 浏览但不能修改文件
cat -n :显示行号
cat 文件名 | more  ：分页显示

less 根据显示需要加载内容，适合大型文件


> ： 将命令显示的内容写入文件，输出重定向，覆盖原先内容
>> ：追加，新的追加到文件末尾

		eg：
		ls -l > a.txt

		ls -l >> a.txt

cat 文件1 > 文件2 将文件1覆盖文件2

echo 输出内容到控制台

 echo $PATH 输出环境变量

 echo "hello world" >> a.txt ：将内容写入文件
 
 
 cal 显示日历
 
head：显示文件开头部分前10行
 
head -n 5 ：前5行
 
tail 文件尾部部分10行

tail -n 5 ：查看倒数5行

tail -f 实时追踪文件的更新
 
ln ：软链接
ln -s 源文件或目录 目标位置
 
（使用 rm -rf 删除软连接）


history：查看历史执行过的指令
history 10 ：最近执行过的10个指令

!指令编号 ： 执行编号是几的指令
